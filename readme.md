
# Тестовое задание для вакансии в Buzzweb

Описание <https://career.habr.com/vacancies/1000042135>

## Используется

- Vue
- Vuex
- Vue Router
- Element UI
- Webpack

## Запуск приложения

- dist/index.html

``` bash
# Установка зависимостей
npm install

# Запуск в режиме development
npm run dev

# Сборка для  production
npm run build

```
